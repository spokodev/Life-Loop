# 01. Product Vision

## Working name
Life-Loop

## Product definition
Life-Loop is a local-first media archival system with verified replicas, optional online access, and explicit lifecycle state for media assets.

## Problem
People generate more photos and videos than they can safely manage. Consumer sync products hide critical state, blur the line between “uploaded” and “safe”, and make cleanup anxious and error-prone.

## Outcome
A user should always know:
- what is new
- what is staged
- what is archived
- what is replicated
- what is verified
- what is safe to remove from phone
- what remains available online

## Product promise
Many ways in. One archive truth.

## Non-goals for MVP
- not a generic cloud drive
- not a social photo app
- not a peer-to-peer torrent system
- not a full DAM for teams on day one

## User archetypes
1. Power user with iPhone + Mac + external drives
2. Hybrid user with local archive + online selected media
3. Advanced user with bring-your-own storage
