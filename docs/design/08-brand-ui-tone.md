# Brand and UI Tone

## Desired product feeling
Life-Loop should feel:
- calm
- trustworthy
- premium utility
- technically competent
- modern without being flashy
- helpful without being paternalistic

## Not this
Life-Loop should not feel:
- crypto-dashboard noisy
- playful consumer-cloud vague
- sterile enterprise bureaucracy
- over-excited startup marketing inside the app
- AI-magic before trust is earned

## Voice rules
- explain clearly
- avoid jargon unless the user is in an explicitly technical area
- never overstate safety or certainty
- use consistent status language across UI and docs
- confirm what is safe, what is pending, and what needs attention

## Visual tone rules
- restrained depth
- refined spacing
- strong hierarchy
- motion used intentionally, not decoratively
- premium through consistency and quality, not excess effects

## Brand posture for future AI
If AI is added later, it should feel like:
- a careful explainer
- a recovery assistant
- a topology helper
- not an autonomous operator
