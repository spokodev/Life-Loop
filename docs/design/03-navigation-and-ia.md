# Navigation and Information Architecture

## Primary sections
- Overview
- Library
- Devices
- Storage
- Activity
- Restore
- Settings

## Rules
- overview answers “am I safe?”
- library answers “what exists?”
- storage answers “where does it live?”
- activity answers “what changed?”
- restore answers “what can I get back?”
- settings answers “what policies apply?”

## Anti-patterns
- burying replica health inside settings
- mixing billing and archive health on one screen
- mixing onboarding with advanced admin settings
